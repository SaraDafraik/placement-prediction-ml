# placement-prediction-ml

A machine learning project that predicts student placement outcomes 
based on academic and skill-related features.

**Built with:** Python, Scikit-learn, Pandas, Google Colab  
**Models used:** Decision Tree, Random Forest  
**Author:** [Sara Dafraik] | BTech CSE (Data Science), 2nd Year

# What it does
- Cleans and preprocesses placement dataset
- Trains and compares Decision Tree and Random Forest classifiers
- Evaluates model performance using accuracy and classification metrics

# Key learnings
- Hands-on experience with supervised ML classification
- Understanding of ensemble methods vs single decision trees
- End-to-end ML pipeline from raw data to model evaluation
